# AIW-Tool

## Brief

This project is used for AIW module.

## Setup

Please follow the below steps to setup:

1. `cd ./The_Folder_To_Find_AiwTool-XXX/driver/`

1. `sudo ./setup.sh`

After setup, the module will obtain IP address automatically while system booting.

Note: please make sure your platform has capability to connect to internet by ethernet during setup procedure.

## Dial-out

1. `cd ./The_Folder_To_Find_AiwTool-XXX/driver/`

1. `sudo ./dialout.sh`

The user can run this script to dial out manually.

## AIW tool

1. `cd ./The_Folder_To_Find_AiwTool-XXX/tool/`

1. `sudo ./tool.sh`

The user can run this script to get/set to this module.
